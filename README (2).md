# 🤖 Jasur Arabic Bot — دليل التشغيل الكامل

A Telegram bot that assesses Arabic language skills, determines CEFR levels, and gives detailed feedback in Arabic. Supports both **written text** and **voice messages**.

---

## 📋 ما الذي يفعله البوت؟

| الميزة | التفاصيل |
|--------|----------|
| ✍️ تقييم الكتابة | الطالب يرسل نصاً عربياً، البوت يحلله |
| 🎤 تقييم الكلام | الطالب يرسل رسالة صوتية، البوت يحولها ويحللها |
| 📊 مستوى CEFR | يحدد المستوى: A1 / A2 / B1 / B2 / C1 / C2 |
| 💬 تغذية راجعة | تقرير مفصل باللغة العربية |
| 🌟 تشجيع | نصائح وتمارين للتطوير |

---

## 🚀 خطوات الإعداد (بدون خبرة برمجية!)

### الخطوة 1: احصل على Telegram Bot Token

1. افتح Telegram وابحث عن **@BotFather**
2. أرسل له: `/newbot`
3. اختر اسماً للبوت (مثلاً: `Jasur Arabic Assessment`)
4. اختر اسم مستخدم ينتهي بـ `bot` (مثلاً: `jasur_arabic_assess_bot`)
5. سيعطيك **Token** يبدو هكذا: `7123456789:AAFxxxxxxxxxxxxxxxxxxxxx`
6. **احفظ هذا الرمز — لا تشاركه مع أحد!**

---

### الخطوة 2: احصل على Gemini API Key

1. اذهب إلى: **https://aistudio.google.com/**
2. سجّل الدخول بحساب Google
3. اضغط **"Get API Key"** في الزاوية اليسرى
4. اضغط **"Create API key"**
5. انسخ المفتاح (يبدو هكذا: `AIzaSyXXXXXXXXXXX`)
6. **احفظه — لا تشاركه مع أحد!**

---

### الخطوة 3: أنشئ حساباً على GitHub

> نحتاج GitHub لرفع الكود على Railway.

1. اذهب إلى **https://github.com**
2. اضغط **"Sign up"** وأنشئ حساباً مجانياً
3. تحقق من بريدك الإلكتروني

---

### الخطوة 4: ارفع الكود على GitHub

**الطريقة السهلة (بدون Terminal):**

1. في GitHub، اضغط **"+"** ثم **"New repository"**
2. سمّه: `jasur-arabic-bot`
3. اجعله **Private** (سري)
4. اضغط **"Create repository"**
5. اضغط **"uploading an existing file"**
6. ارفع جميع ملفات المشروع:
   - `bot.py`
   - `requirements.txt`
   - `railway.toml`
   - `Procfile`
   - `.gitignore`
   - ⚠️ **لا ترفع ملف `.env`** — يحتوي على مفاتيح سرية!
7. اضغط **"Commit changes"**

---

### الخطوة 5: انشر على Railway

1. اذهب إلى **https://railway.app**
2. اضغط **"Start a New Project"**
3. اختر **"Deploy from GitHub repo"**
4. وصّل حساب GitHub وارفع مستودعك
5. سيبدأ Railway في بناء المشروع تلقائياً

---

### الخطوة 6: أضف المتغيرات السرية في Railway

هذه الخطوة **مهمة جداً** — هنا نضع المفاتيح السرية بدلاً من ملف `.env`:

1. في لوحة Railway، اضغط على مشروعك
2. اضغط على **"Variables"** في القائمة الجانبية
3. أضف المتغيرَين التاليَين:

```
TELEGRAM_BOT_TOKEN = الرمز الذي أعطاكه BotFather
GEMINI_API_KEY     = مفتاح Gemini الذي أنشأته
```

4. اضغط **"Add"** بعد كل متغير
5. Railway سيعيد تشغيل البوت تلقائياً

---

### الخطوة 7: تحقق من تشغيل البوت

1. في Railway، اضغط **"Deployments"** لترى سجل التشغيل
2. ابحث عن هذا السطر في اللوق:
   ```
   ✅ Bot is running. Waiting for messages...
   ```
3. افتح Telegram وابحث عن اسم البوت الذي أنشأته
4. أرسل `/start` وجرّبه!

---

## 🧪 كيفية اختبار البوت

### اختبار الكتابة:
1. أرسل `/start`
2. اضغط **"✍️ تقييم الكتابة"**
3. أرسل نصاً عربياً مثل:
   > "ذهبت إلى السوق وأشتريت خضروات طازجة لتحضير وجبة العشاء مع عائلتي."
4. انتظر التقرير (يستغرق 5-15 ثانية)

### اختبار الكلام:
1. أرسل `/start`
2. اضغط **"🎤 تقييم الكلام"**
3. أرسل رسالة صوتية بالعربية
4. انتظر: أولاً سيظهر النص المحوَّل، ثم التقرير
